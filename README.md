# Power BI Data Refresh Mechanism Project

This project demonstrates how to implement automatic data refresh in Power BI.

## Project Goals
- Understand the difference between Import Mode and DirectQuery
- Publish reports to Power BI Service
- Configure Scheduled Refresh
- Set up On-Premises Data Gateway

## Tools Used
- Power BI Desktop
- Power BI Service
- On-premises data gateway

## Features
- Interactive dashboard
- Automated dataset refresh
- Gateway configuration
